# AI Fairness Auditor — FastAPI Backend

## Setup
```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

## Endpoints
| Method | Route      | What it does                                      |
|--------|------------|---------------------------------------------------|
| GET    | /          | Health check                                      |
| POST   | /upload    | Upload CSV → trains all 3 models → returns audit  |
| GET    | /audit     | Fairness metrics for all 3 models as JSON         |
| GET    | /summary   | Same as /audit (alias for dashboard)              |
| POST   | /predict   | Single applicant → decision from all 3 models     |
| GET    | /status    | Is the model trained and ready?                   |

## Interactive docs
http://localhost:8000/docs   ← auto-generated by FastAPI, test all endpoints here

## Usage flow
1. POST /upload  with your CSV file
2. GET  /audit   to see bias metrics
3. POST /predict with individual applicants to compare decisions
